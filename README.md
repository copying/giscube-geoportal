# Giscube Geoportal

> Giscube Geoportal

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
quasar dev

# build for production with minification
quasar build

```

## Publish library

Prerelease:

npm version prerelease --preid=beta

LIB=True quasar build

npm publish
